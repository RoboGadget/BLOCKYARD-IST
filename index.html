How to use

  1. Add blocks from above
  2. Drag the label bar to move — drop onto a Container to nest inside it
  3. Drag the corner to resize
  4. Click a block to set its name, size, color, padding, and margin
  5. Hit HTML/CSS ME when done

Select a block on the canvas to edit its name, position, size, and color.
